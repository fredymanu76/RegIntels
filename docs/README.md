# RegIntels Documentation

Comprehensive documentation for the RegIntels compliance and governance platform.

## Folder Structure

### `/architecture`
System architecture and database schema documentation:
- `ARCHITECTURE.md` - Overall system architecture
- `DATABASE_SCHEMA_REFERENCE.md` - Complete database schema reference
- `EXISTING_SCHEMA_REFERENCE.md` - Legacy schema documentation

### `/deployment`
Deployment guides and solution rollout documentation:
- Solution 4 deployment (Risk Signal Hub, Strategic Scoring)
- Solution 5 deployment (Board-level dashboards)
- Tenant management deployment
- Platform feature deployment guides

### `/migration-guides`
Data migration and upgrade instructions:
- Compliance module migration
- LMS platform migration
- Migration checklists and quick-start guides

### `/setup-guides`
Initial setup and configuration guides:
- Platform admin setup
- Password reset configuration
- Feature control setup
- Troubleshooting guides

### `/session-notes`
Development session notes and implementation logs:
- Feature implementation notes
- Bug fix summaries
- Dashboard integration notes

## Key Documents

- **Start Here**: Check `/setup-guides/START_HERE_*.md` files for getting started
- **Deployment**: See `/deployment` for production rollout instructions
- **Troubleshooting**: See `/setup-guides/TROUBLESHOOTING.md` for common issues

## For Developers

1. Review `ARCHITECTURE.md` for system overview
2. Check `DATABASE_SCHEMA_REFERENCE.md` for database structure
3. Follow deployment guides in chronological order for new features
